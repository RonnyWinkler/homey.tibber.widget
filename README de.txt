Diese App erweitert dein Homey Dashboard um ein praktisches Widget für die Tibber App (ab Version 1.10.0). 

Das Widget zeigt ein Diagramm der aktuellen Strompreise sowie den Live-Verbrauch deines Tibber Pulse Stromzählers. 
So behältst du deinen Energieverbrauch und die Stromkosten jederzeit im Blick – direkt auf deinem Homey Dashboard!
